# world-of-airports

## Running the app
Both server and client needs to be running.
Prerequisite: Node (8+ needed) installed

### Run Server:
In a terminal window:
1. `cd server`
2. `npm install`
3. `npm run start`
Keep terminal open.

### Run client:
In new terminal window
1. `cd client`
2. `npm install`
3. `npm run start`
4. open localhost:3000 in browser
